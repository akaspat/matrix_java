# matrix_java
В целочисленной матрице А размером N*M заменить элементы побочной диагонали и расположенные выше нее на значение 4 (числа N и М задаются с клавиатуры в диапазоне от 3 до 10). Первоначальное заполнение массива А осуществить при помощи датчика случайных чисел в диапазоне от -5 до 15. Исходную и видоизмененную матрицы вывести на экран.
