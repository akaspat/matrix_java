# matrix_java
В целочисленной матрице А размером N*M заменить элементы побочной диагонали и расположенные выше нее на значение 4  
(числа N и М задаются с клавиатуры в диапазоне от 3 до 10).  
Первоначальное заполнение массива А осуществить при помощи датчика случайных чисел в диапазоне от -5 до 15.  
Исходную и видоизмененную матрицы вывести на экран.  

* Предложить 2 варианта, (назовем исходники – Код1 и Код2) реализации
задания в рамках:
    * структурного программирования
_код1 - (вся реализация задания внутри main()) (см. репо: [matrix_cpp](https://github.com/akaspat/matrix_cpp) )_  
    * объектно-ориентированного программирования  
_код2 (cм. репо: [matrix_java](https://github.com/akaspat/matrix_java))_  

* Оформить код программы с построчной нумерацией для последующего анализа
* Произвести полный анализ программ (код1 и код2 по отдельности) метрикой
Холстеда, в т.ч. оценив предположительное число ошибок, заполнив четыре
таблицы. Сделать содержательный вывод.
    * Словарь операций и операторов программы (отдельно для код1 и код2)
    * Словарь операндов программы (отдельно для код1 и код2)
    * Входные и выходные переменные программы (отдельно для код1 и код2)
    * Сводная таблица характеристик Холстеда, включая время разработки и
количество ошибок (единая для код1 и код2 – соответственно столбец
«значение для код1» и столбец «значение для код2» )  

* Произвести оценку программ (код1 и код2 по отдельности) метрикой Джилба.
Сделать содержательный вывод.
* Произвести оценку программ (код1 и код2 по отдельности) метрикой Чепина.
Сделать содержательный вывод.
* Произвести оценку программ (код1 и код2 по отдельности) структурной
сложности программ (критерий 1, критерий 2, критерий 3). Для этого
построить как граф потока управления, так и граф потока данных), по
возможности, укрупняя узлы графов. Сделать содержательный вывод.
_Примечание: допускается в случае малой информативности матрицу
достижимости не предоставлять._

* Произвести оценку программ (код1 и код2 по отдельности) метрикой МакКейба.
_Сделать вывод о том, насколько метрика МакКейба информативнее (или нет?)
критерия_1, критерия_2 (есть ли отличие ?) , критерия_3._
